Viewpoint

Viewpoint is a specification of the conventions for a particular view kind/type. Views are instances of viewpoints.

LinkedIn profile viewpoint specifies that a profile may contain “Profile photo” representation and “Experience” representation.

In the above example of a web page view, the viewpoint may specify that a view shall contain an intro paragraph, a component diagram, an optional sequence diagram and then either a list or sub-sections with short descriptions of components.
