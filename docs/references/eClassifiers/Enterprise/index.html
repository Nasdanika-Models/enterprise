Enterprise

One definition of an enterprise is “A group of people and resources devoted to achieving a mission.”1 We will use the term “stakeholder” instead of “people”. A stakeholder can be a person, an organization/another enterprise, or an automated agent, including AI agents.

Note that an enterprise may have a single stakeholder. Going fishing alone or with friends is an enterprise. Writing a book is also an enterprise.

An enterprise operates on one or more semantic domains. For example, the book domain - author, reader, publisher, book, chapter, …

Enterprises can be nested and interrelated. A classic example is a company with departments depending on each other. This model does not reflect this aspect for brevity.
