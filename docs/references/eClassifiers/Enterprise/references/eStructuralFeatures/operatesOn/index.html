Operates on

An enterprise operates on one or more semantic domains. For example, an Agile team developing Spring microservices to be deployed to Kubernetes operates on at least three semantic domains - Agile (stories, sprints, …), Spring (controllers, …), and Kubernetes (deployments, images, …)
